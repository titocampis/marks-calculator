# Marks Calculator for a UOC degree

Example of json to load:
```json
{
  "Assignatura 1": {
    "marks": [5, 7, 5, 10],
    "wages": [11, 22, 27, 39]
  },
  "Assignatura 2": {
    "marks": [18, 15, 20, 30],
    "wages": [20, 20, 20, 40]
  }
}
```

Or different credit size subjects:
```json
{
  "Assignatura 1": {
    "marks": [5, 7, 5, 10],
    "wages": [11, 22, 27, 39],
    "credits": 5,
  },
  "Assignatura 2": {
    "marks": [18, 15, 20, 30],
    "wages": [20, 20, 20, 40],
    "credits": 10
  }
}
```
